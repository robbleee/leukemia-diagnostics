WHO Haematologic Classification Tool - Major Functionality

1. Core Features
---------------
- Web-based application for classifying haematologic malignancies
- User authentication system with secure password hashing
- Interactive data entry forms
- Real-time classification results
- PDF report generation
- AI-powered clinical recommendations (for authenticated users)

2. Classification Capabilities
----------------------------
- Supports classification of 26 different haematologic malignancies including:
  * Acute Myeloid Leukemia (AML) variants
  * Acute Lymphoblastic Leukemia (ALL)
  * Myelodysplastic Syndromes (MDS)
  * Various Lymphoma types
  * Other rare blood cancers

3. Risk Assessment Tools
-----------------------
- ELN 2022 Risk Assessment for AML
- ELN 2024 Non-Intensive Risk Assessment
- IPSS Risk Assessment for MDS
- IPSS-M Risk Assessment
- Dynamic risk visualization and interpretation

4. Data Input Methods
--------------------
- Manual data entry forms
- Report parsing capabilities:
  * Genetics report parsing for AML
  * Genetics report parsing for MDS
  * IPSS report parsing
  * ELN report parsing

5. Results and Analysis
----------------------
- Detailed classification results
- Step-by-step derivation explanations
- Interactive flowcharts
- Risk stratification visualization
- Treatment recommendations
- Clinical next steps
